# SMBexcel `.github`

This repository powers the [SMBexcel organization](https://github.com/SMBexcel) landing page.

GitHub's convention: any markdown file placed at `profile/README.md` inside a special repository named `.github` under an organization is rendered as that organization's public profile page.

The org landing content lives at [`profile/README.md`](./profile/README.md).

---

## Don't install this

There's nothing to install or fork here — it's purely the org's front-door content. For actual skills and tools, see [SMBexcel/skills](https://github.com/SMBexcel/skills).
